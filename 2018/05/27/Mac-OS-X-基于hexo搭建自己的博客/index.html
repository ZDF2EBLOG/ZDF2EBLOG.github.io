Mac OS X 基于hexo搭建自己的博客

发表于 2018-05-27

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。
更多查看： https://hexo.io/zh-cn/docs/

一、安装hexo


sudo npm install -g hexo

二、创建文件夹


mkdir firstBlog
cd firstBlog
hexo init

之后得到如下目录


·
|– _config.yml
|– package-lock.json
|– package.json
|– scaffolds
|– source
|   |– _posts
|– themes

三、生成静态网页


hexo generate

四、在服务器上运行


hexo server

假如你想指定端口可以使用下面命令
hexo server -p 端口号
或者 hexo s -p 端口号

五、撰写博客


hexo new post “博客名字”

成功后出现


INFO  Created: ~/Desktop/博客/firstBlog/source/_posts/’博客名字‘.md
